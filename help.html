FAQよくある質問

このサービスは誰でも使えますか？
GOLF Create 戸津池店の会員様専用のサービスです。
アプリのインストールは必要ですか？
不要です。お手持ちのLINEから、そのままご利用いただけます。App StoreやGoogle Playでの検索は必要ありません。
打席予約システムから、コーチも予約できますか？
いいえ。コーチのご予約は、この MyCoach（LINE）で行います。いつもの打席予約システムからはコーチのご予約はできません。打席のご予約は GOLF Create で、コーチのご予約は MyCoach で、それぞれ必要です。
支払いは、打席予約に登録したカードで行われますか？
いいえ。MyCoach でのお支払いには、MyCoach に改めてクレジットカードのご登録が必要です。打席予約システムに登録されたカードは使われません。
料金はどこで分かりますか？
料金は各コーチのページに表示されます。ご予約のお手続きの中でも金額をご確認いただけます。
コーチはどう選べばいいですか？
各コーチのプロフィール・実績・口コミ評価を見て、ご自身に合うコーチをお選びください。気になるコーチはお気に入りに登録できます。
予約の確認・キャンセルはどこから？
ご予約の確認は、マイページの「予約一覧」からご覧いただけます。キャンセル・変更は予約一覧からお手続きいただくか、このLINE・GOLF Create 戸津池店へご相談ください。返金率は上記「キャンセル・変更について」をご確認ください。
ログインできないときは？
LINEでもう一度ログインをお試しください。解決しない場合は、このLINEにメッセージをお送りください。順番にご対応いたします。

お困りのときは

このLINEにメッセージをお送りください。
GOLF Create 戸津池店 がご対応いたします。
