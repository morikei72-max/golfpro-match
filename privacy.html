株式会社ケイエスケー（以下「当社」といいます）は、MyCoach（以下「本サービス」といいます）をご利用いただくユーザーの皆様の個人情報を、個人情報の保護に関する法律（個人情報保護法）その他関連法令を遵守し、適切に取り扱います。本プライバシーポリシーは、本サービスにおける個人情報の取り扱いについて定めるものです。

第1条（事業者情報）

本サービスの運営事業者は以下のとおりです。

  • 事業者名：株式会社ケイエスケー
  • 所在地：京都府八幡市
  • サービス名：MyCoach（マイコーチ）
  • 個人情報保護管理責任者：代表取締役

第2条（取得する個人情報）

本サービスでは、お客様・コーチ・店舗管理者の各区分に応じて、以下の個人情報を取得する場合があります。

2-1. お客様から取得する情報

  • 氏名、ふりがな、メールアドレス、電話番号
  • 年齢層、ゴルフ歴、ハンディキャップ、所属ゴルフ場
  • Golf Create 戸津池店の会員番号・会員種別（会員認証時）
  • 予約履歴、レッスン内容、決済情報、レビュー投稿
  • LINE連携時の LINE ユーザーID（任意）

2-2. コーチから取得する情報

  • 氏名、住所、生年月日、電話番号、メールアドレス
  • 身分証明書画像（運転免許証・マイナンバーカード等）
  • 保有資格、自己紹介、得意分野、プロフィール写真
  • インボイス登録番号（登録されている場合のみ）
  • 本人確認書類、銀行口座情報（報酬支払のため・決済代行事業者経由）
  • レッスン履歴、収入情報、LINE ユーザーID（任意）

2-3. 店舗管理者から取得する情報

  • 氏名、店舗名、連絡先、メールアドレス

2-4. 自動的に取得される情報

  • IPアドレス、ブラウザ種類、アクセス日時、閲覧ページ情報
  • Cookie および類似技術によって取得される情報

第3条（個人情報の利用目的）

当社は、取得した個人情報を以下の目的で利用します。

  1. 本サービスの提供、運営および改善
  2. お客様とコーチのマッチング、予約管理、決済処理
  3. ユーザー認証、本人確認、会員資格の確認
  4. レッスンに関するご案内、通知、リマインダーの送信
  5. お問い合わせ・サポート対応
  6. コーチへの報酬支払、店舗への手数料精算
  7. 利用規約違反、不正利用、なりすまし等の防止・調査
  8. 本サービスの改善、統計データの作成（個人を特定しない形式）
  9. 法令に基づく対応

第4条（第三者提供）

当社は、次のいずれかに該当する場合を除き、お客様の同意を得ずに個人情報を第三者に提供することはありません。

  1. 法令に基づく場合
  2. 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
  3. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
  4. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合
  5. 本サービスの運営上、業務の一部を外部委託する場合（後述の第5条参照）
📌 コーチ・お客様間の情報共有について
予約成立時、コーチにはお客様の氏名・連絡先が通知されます。また、お客様がコーチをお気に入り登録された場合、コーチにお客様の氏名が通知されます（利用規約第4条に準拠）。これらは本サービスの性質上必要な情報共有であり、第三者提供には該当しません。

第5条（業務委託先への提供）

当社は、本サービスの運営のため、以下の業務委託先に個人情報の取扱いを委託する場合があります。委託先とは機密保持契約を締結し、適切な監督を行います。

委託先区分委託内容
決済代行事業者クレジットカード決済の処理、コーチへの報酬送金
メッセージ配信事業者LINE連携によるメッセージ配信
クラウドサーバー事業者本サービスのデータ保管、システム運用
メール配信事業者予約通知・認証メールの送信
💳 決済情報の取り扱いについて
クレジットカード番号等の決済情報は、PCI DSS 準拠の外部決済サービスにおいて直接処理され、当社のサーバーには保存されません。当社が取得・保持するのは取引IDおよび金額情報のみです。

第6条（安全管理措置）

当社は、個人情報の漏洩、滅失、毀損等を防止するため、以下の安全管理措置を講じています。

  • 組織的安全管理措置：個人情報保護管理責任者の設置、取扱規程の整備
  • 人的安全管理措置：従業員への教育、秘密保持契約の締結
  • 物理的安全管理措置：データセンターへの入退室管理、機器の盗難防止
  • 技術的安全管理措置：SSL/TLS暗号化通信、アクセス制御、不正アクセス対策

身分証データの取り扱い

コーチから提供された身分証明書画像は、本人確認およびなりすまし防止の目的のみに使用し、暗号化して保管します。コーチが本サービスを退会された場合、退会後30日以内に完全に削除します。

第7条（Cookie および類似技術）

本サービスでは、ユーザー体験の向上および利用状況の分析のため、Cookie および類似技術を使用する場合があります。

  • ログインセッションの維持
  • ユーザー設定の保存
  • アクセス解析（個人を特定しない統計情報）

Cookieの受け入れはブラウザ設定で無効にすることができますが、その場合、本サービスの一部機能がご利用いただけない可能性があります。

第8条（LINE連携について）

本サービスでは、予約通知・各種お知らせの配信手段としてLINE公式アカウント「MyCoach」との連携機能を提供しています。

  • LINE連携は完全に任意であり、連携しなくても本サービスの基本機能をご利用いただけます。
  • 連携時、お客様のLINEユーザーIDを取得し、当社のサーバーに保管します。
  • 取得したLINE情報は、予約通知・本サービスに関するご案内以外の目的では使用しません。
  • LINEとの連携はいつでも解除できます。

第9条（Golf Create 戸津池店 会員情報の扱い）

本サービスでは、Golf Create 戸津池店の会員のみがお客様として登録できます。会員認証のため、Golf Create側が保有する会員情報（氏名・メールアドレス・会員番号等）と照合を行います。

  • Golf Create 戸津池店を退会された場合、本サービスのご利用は停止されます。
  • 退会後、本サービスに保管されている個人情報は、法令で定める保存期間経過後、速やかに削除します。
  • 再入会された場合は、再度会員認証が必要です。

第10条（本人開示・訂正・利用停止等の請求）

お客様は、当社が保有するご自身の個人情報について、以下の請求を行うことができます。

  1. 利用目的の通知
  2. 開示
  3. 内容の訂正、追加または削除
  4. 利用の停止または消去
  5. 第三者提供の停止

ご請求にあたっては、本人であることを確認させていただきます。ご本人以外からのご請求にはお応えできません。詳細は第13条のお問い合わせ窓口までご連絡ください。

第11条（未成年者の個人情報）

本サービスは、原則として成年（18歳以上）のご利用を想定しています。未成年者が本サービスをご利用になる場合は、必ず保護者の同意を得てください。未成年者からの個人情報取得が判明した場合、保護者のご同意を確認のうえ、必要に応じて削除等の対応を行います。

第12条（プライバシーポリシーの改定）

当社は、法令の改正、本サービスの内容変更、その他必要に応じて、本プライバシーポリシーを改定することがあります。重要な変更があった場合は、本サービス上で事前に告知します。改定後のプライバシーポリシーは、本サービス上に掲載された時点から効力を有するものとします。
